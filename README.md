# YONGRYEON Gallery - Cloudflare Pages Step 1

실제 배포 가능한 1단계 완성형 정적 전시장입니다.

## 포함 기능
- 첫 화면: YONGRYEON
- Works
- About
- Contact
- 샘플 작품 없음
- 관리자 등록 작품만 표시할 수 있도록 API 연결 자리 준비
- Cloudflare Pages 바로 배포 가능

## Cloudflare Pages 배포 방법
1. Cloudflare 로그인
2. Workers & Pages → Create application → Pages
3. Upload assets 선택
4. 이 폴더 안의 파일 전체를 업로드
5. Deploy 클릭

## GitHub Pages 배포 시 주의
index.html이 저장소 가장 바깥(root)에 있어야 합니다.
