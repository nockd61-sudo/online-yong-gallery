# 세웅 3D 온라인 미술관 v2.0

## 사용 방법
1. GitHub 저장소에 이 파일들을 업로드합니다.
2. GitHub Pages로 접속합니다.
3. `admin.html`로 들어갑니다.
4. 초기 비밀번호는 `1234`입니다.
5. 대표 이미지와 GLB 파일을 선택하고 작품을 등록합니다.

## 중요
이 기본형은 브라우저 저장 방식(localStorage)입니다.
같은 PC/브라우저에서는 작품이 유지되지만, 모든 방문자에게 공통으로 저장하려면
Cloudflare R2 / Supabase / Firebase 같은 서버 저장소 연결이 필요합니다.

## 다음 단계
완전한 운영형으로 가려면:
- 관리자 로그인 보안 강화
- Cloudflare R2에 이미지/GLB 저장
- 작품 데이터 DB 저장
- GitHub를 거치지 않는 실시간 등록 구조
